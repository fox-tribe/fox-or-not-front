## 🦊 "너 그거 폭스야"
#### “주변인의 이 행동 너무 헷갈려요… 오해일까요?”🤔 알쏭달쏭한 이성의 행동에 답을 알려주는 커뮤니티 만들기😽"

## 목표
-  머신러닝을 활용하여 커뮤니티 내 필터링 적용하기
-  GitHub 활용 및 효율적인 분업
-  DRF를 활용하여 실력 높이기
-  Docker를 활용하여 배포하기
-  CRUD 기능 개발하기 
-  배포 후 커뮤니티 등의 홍보를 통해서 실제 사용자를 모아 피드백 받아보며 개선하기
